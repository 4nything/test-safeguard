Add the User's Current Temperature to the Header

Get the user's location (latitude & longitude) based on their browser and search the openweathermap.org free weather api to pull the current temperature in fahrenheit for that location.

Use the OpenWeatherMap API KEY: 354b4e751f41872ac5ee002579748a47

There are a few requirements for this feature:

  • The temperature should be loaded as soon as the website is opened by the user and should always be visible as the user navigates the site.
  • The temperature should be displayed on the right side of the header, near the github logo.
  • The temperature should be updated every 30 seconds.
  • The temperature should be rounded to the nearest degree.
  • The temperature should be displayed with the degree ยบ symbol.
