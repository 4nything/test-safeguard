Publish your test to a (free) GitHub Pages Site

Once you are done with all the tasks, follow the steps below to set up your Developer Test on GitHub Pages.

  1. Commit and push your code.
  2. Deploy the Angular App to your gh-pages branch by running:
    ng deploy
    This command automatically creates the gh-pages branch and pushes it to your GitHub repo. You may need to authenticate with GitHub if you are not already.
