Determine the Smallest Missing Positive Integer

Using the SmallestIntegerComponent, write an efficient function smallestMissingInt(A) that will determine the smallest missing integer (greater than 0), given an array A of N integers.

For example:

Given A = [1, 2, 3, 4, 5] the function should return 6.

Given A = [1, 6, 3, 4, 2, 1] the function should return 5.

Given A = [-1, -6, 4] the function should return 1.

You can make a few assumptions:

  • N is an integer within the range of [1 ... 100,000]
  • each item of array A is an integer within the range of [-1,000,000 ... 1,000,000]
